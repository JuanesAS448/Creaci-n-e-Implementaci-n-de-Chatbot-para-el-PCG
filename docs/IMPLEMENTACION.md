# UNI SIGEA - Implementación

Proyecto implementado en HTML, CSS y JavaScript vanilla.

## Puesta en marcha
1. Abrir `index.html` en navegador moderno.
2. Configurar `config/env.js` con la API key de Gemini.
3. Validar navegación lateral, secciones dinámicas y chatbot.
